# Contributing

## Development

First clone the project and then run `npm link`.

```bash bash
npm ci
npm link
```

You should now be able to run the bin scripts.


## Documentation

Generate docs with [markatzea][markatzea].

```bash bash
markatzea README.mz > README.md
```
